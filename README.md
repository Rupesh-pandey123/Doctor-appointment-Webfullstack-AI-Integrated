The Doctor Appointment Booking System is a full-stack web application designed to simplify the process of booking medical appointments. It allows patients to easily find doctors, view available time slots, and schedule appointments online without hassle.
# 🩺 Doctor Appointment Booking System (AI Integrated)

An advanced full-stack web application that simplifies healthcare access by enabling patients to book doctor appointments online. This system is enhanced with AI-powered assistance, secure online payments, and digital prescription management.

---

## 🌟 Features

### 👤 Patient Features

* User Registration & Login (JWT Authentication)
* Search Doctors by specialization, location, availability
* Book, Cancel, or Reschedule Appointments
* Secure Online Payment Integration
* View Appointment History
* Access Digital Prescriptions

### 🧑‍⚕️ Doctor Features

* Doctor Profile Management
* Availability & Time Slot Management
* View & Manage Appointments
* Generate and Upload Digital Prescriptions

### 🤖 AI Integration

* AI-based symptom checker
* Smart doctor recommendation system
* Chatbot for basic health queries

### 💳 Payment Integration

* Secure payment gateway integration (Razorpay/Stripe)
* Payment status tracking
* Invoice generation

### 📄 Prescription System

* Doctors can upload prescriptions
* Patients can download/view prescriptions anytime
* Secure storage of medical records

### 🛠️ Admin Panel

* Manage Users & Doctors
* Monitor Appointments
* Manage Payments & Reports
